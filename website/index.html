Quick Start

1

Start LLMFlow

npx llmflow

Or: docker run -p 3000:3000 -p 8080:8080 helgesverre/llmflow

2

Point your SDK

# Python
client = OpenAI(base_url="http://localhost:8080/v1")

# JavaScript  
const client = new OpenAI({ baseURL: 'http://localhost:8080/v1' });
3

View your traces

Open localhost:3000 to see costs, tokens, and latency.
